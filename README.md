# 🏢 HR Management System Frontend

Welcome to the **HR Management System Frontend**! This is the powerhouse UI that drives our HR solution, built with modern web technologies to be fast, scalable, and a joy to use. 🚀

## ✨ What's Inside?

We've packed this system with everything you need to manage a modern workforce. No "star" features here because every module is built to shine:

- 🔐 **Auth & Security:** Robust authentication and page-level permissions.
- 👥 **Employee Management:** Full lifecycle tracking for your most valuable assets.
- 💼 **HR Operations:** Streamlining day-to-day human resources tasks.
- 💰 **Payroll:** Managing compensation, benefits, and everything in between.
- 🏢 **Organizational structure:** Mapping out your company's DNA.
- ⚙️ **System Settings & Look-ups:** Fine-grained control over how the system behaves.
- 📅 **Decisions & Workflows:** Handling the logic that keeps the business moving.

### 🪄 The Secret Sauce: Dynamic OData Query Generation
One of our coolest tricks is how we handle data. We use a **Visual React Query Builder** to dynamically generate complex **OData queries**. This means you get a powerful, intuitive interface to filter and find exactly the data you need without writing a single line of query code. It’s like magic, but with better types! 🪄✨

## 🛠️ Tech Stack

We use a modern, type-safe stack to ensure everything runs smoothly:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed for peace of mind)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [MUI](https://mui.com/)
- **Data Fetching:** [TanStack Query](https://tanstack.com/query/latest) (React Query)
- **Components:** [Radix UI](https://www.radix-ui.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Forms:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Query Building:** [React Query Builder](https://react-querybuilder.js.org/)
